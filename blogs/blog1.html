Centering using CSS

There are many ways to position content inside a container in CSS. We can achieve this using multiple way, which are as follows:


Using Transform-Translate

Using THE "OG" WAY i.e Transform, this guarantees that you text is in the center of the container, no matter what.
It is a primitive way to handle positioning but as they say, OLD IS GOLD. This method might get the job done but its least FLEX-ible(pun intended).

Here, .any-content is the target class which you want to position in the center of the container. Making it position: absolute; guarantees that our transformation works but be careful with it.
Lastly, adding top:50%; and left:50%; to force the element to be in center of container no matter what the size of the screen. and then the icing on the cake transform: translate(-50%,-50%); to adjust for element height and width



Using Flex

This is a more user-friendly approach where instead of element, you manipulate your container itself. Using this method makes your container a flexbox and then you can make all elements in a container behave a certain way.

Here display:flex; is used to make your container a flexbox and then you can add direction of flexbox as flex-direction:row; or flex-direction: column; for simplicity, i will use flex-direction: column;.
After configuring direction, just add align-items: center; and justify-content: center; and voila!, you have made all of your content align to center by default and it's more Responsive.

Note: display: flex; has some more Functionalities which can be quite handy but are beyond the scope of this topic so you might want to check this space for more such content :D

So, there you have it! Two easy ways to make your content responsive and allign to center. Happy Coding!
